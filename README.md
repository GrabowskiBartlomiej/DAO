# DAO
workshop2

Second module conclusion excercise - DAO

* usable only via JVM,
* has 4 classes:
    - Main to call all methods;
    - DbUtil for the connection to database;
    - User mirroring table users from database;
    - UserDao with SQLQueries and 4 methods:
    		+Create new user;
    		+Read uses on id;
    		+Update table on id;
    		+Delete user on id;
